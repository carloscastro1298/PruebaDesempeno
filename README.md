# Performance Test
Student Management System (Python CRUD with JSON)

Description

This project is a simple Student Management System built in Python.
It allows you to perform basic CRUD operations (Create, Read, Update, Delete) using a JSON file as storage.

The system is console-based and designed for beginners who are learning Python, file handling, and data structures.

---

Features

- Register new students
- View student list
- Search students by ID or name
- Update student information
- Delete students
- Data persistence using a JSON file

---

Project Structure

project/
│
├── main.py          # Main application file
├── students.json    # Data storage file
└── README.md        # Project documentation

---

Requirements

- Python 3.x

No external libraries are required.

---


---

Usage

Run the program with:

python main.py

Then follow the menu options:

1. Register student
2. List students
3. Exit

Example JSON Structure

[
    {
        "id": "1",

        "age": 12,
        "course": "Math",
        "status": "active"
    }
]
